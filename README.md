# Projet 1

Ce dossier sert a suivre les fonctions a implementer et les fonctions deja ajoutees.

## Workflow
- Ajouter chaque nouvelle idee de fonction dans `TASKS.md`.
- Quand une fonction est implementee, la supprimer de `TASKS.md`.
- Ajouter ensuite la fonction terminee dans `FUNCTIONS.md` avec une courte description.
- Garder les noms de fonctions simples et explicites.

## Fichiers
- `TASKS.md` : fonctions et taches a faire.
- `FUNCTIONS.md` : fonctions deja ajoutees au projet.
